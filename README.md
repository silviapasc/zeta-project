# Zeta-Projekt

Das Zeta-Projekt implementiert einfache Funktionen in Python zur Berechnung von Burrows' Zeta, einer in der Computerlinguistik gebräuchlichen Maßeinheit zum kontrastiven Lesen von Textkorpora.
